# 📚 AI Research Paper Analyzer

AI-powered research paper analysis system that automatically extracts paper information, generates summaries, identifies research gaps, compares papers, and provides reviewer-style insights from uploaded PDF documents.

## Features

- PDF Upload
- Text Extraction
- Title Detection
- Abstract Extraction
- Author Detection
- Research Summary
- Research Gap Analysis
- Paper Comparison
- Reviewer Recommendations

## Technologies

- Python
- Streamlit
- PyMuPDF
- Pandas
- NLTK
- Scikit-Learn

## Run

pip install -r requirements.txt

streamlit run app.py
